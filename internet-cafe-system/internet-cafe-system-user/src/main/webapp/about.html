河南省交通运输厅公路管理局

文明上网管理规定

为进一步改进机关工作作风，严肃机关工作纪律，维护机关工作秩序，规范机关工作行为，坚决禁止利用上班时间炒股、玩游戏等不良行为，营造良好的工作环境，特作如下规定：

1、自觉遵守有关保守国家机密的各项法律规定，不泄露党和国家机密，或传送有损国格、人格的信息。

2、不得发表任何诋毁国家、政府、党的言论，不得发表任何有碍社会稳定、国家统一和民族统一的言论。 

3、不制作、查阅、复制和传播有碍社会治安及社会公德和有伤社会风化的信息。

4、不得使用软件的或硬件的方法窃取他人口令，非法入侵他人计算机系统，阅读他人文件或电子邮件，滥用网络资源。

5、不在网络上接收和散布封建迷信、淫秽、色情、赌博、暴力、凶杀、恐怖等有害信息。不得浏览色情、暴力、反动站点。 

6、不得捏造或歪曲事实、散布谣言、诽谤他人、扰乱社会秩序的不良信息。

7、原则上不允许职工自带电脑、笔记本上班，不允许私拉网线、更改主机IP等。 

8、在上班期间上网不得大声喧哗，不得播放电影、音乐，以免影响其他职工。 

9、禁止在办公时间上网聊天、看电影视频、玩游戏、炒股等。 

10、下班离开办公室前，应该注意重要资料备份，安全关闭电脑并切断电源。 

11、对违反规定的职工给予通告批评，违反信息安全或造成后果的，将依法追究相关责任人的责任，严重者送交公安机关处理。

本规定自发布之日其执行。

河南省交通运输厅公路管理局办公室
